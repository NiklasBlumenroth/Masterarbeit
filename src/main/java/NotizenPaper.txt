2. Scientific Background
2.1. Distributed mental models
- mentales Modell ist die Darstellung eines spezifischen Problems im Kopf einer Person
- in heterogenen Gruppe existieren unterschiedliche Modelle -> verteiltes Mentales Modell (DMM)
- kooperative Gruppe: gleiches Ziel
- rationale Gruppe: unterschiedliche Bewertungen
- unterschiedliche Bewertungen sind Zeichen für unterschiedliche mentale Modelle
- Minderheiteninformationen sind für die Entscheidungskompetenz einer Gruppe von entscheidender Bedeutung

2.2. Use of entropy in MCDM
- Informationsentropie -> Unsicherheit
- Entropiegewichtungsmethode wird verwendet, um Kriteriengewichte zu bestimmen, ohne Informationen von den Entscheidungsträgern zu verwenden
- Kriterien mit geringer Variabilität tragen wenig zur Unterscheidung der Alternativen bei
- Kriterien mit geringer Entropie wird höhere Wichtung zugewiesen
- Entscheidungsträgern, deren Bewertungen eine größere Unterscheidungskraft aufweisen, wird ein höheres Gewicht zugewiesen
- Informationseentropie für Probleme ohne Kriterieninformationen
- Entscheidungsträger wird wiederholt aufgefordert welche von zwei Alternativen er bevorzugt -> Raum zulässiger Kriteriengewichte zu reduzieren
- Ziel: Aufforderungen minimieren (Austausch minimieren)

2.3. Consensus-building in MCDM
- hier ist Konsens eine numerische Größe (Einstimmigkeit)
- Weicher Konsens bedeutet, dass das berechnete Maß an Übereinstimmung erreicht wird
- völlige Einstimmigkeit gilt als nicht erreichbar
- das Zusammenfassen oder Weglassen von Eingaben(Bewertungen) verhindert zu einem Konsens zu gelangen

2.4. Stochastic multicriteria acceptability analysis
- tool zur Analyse von Gruppenentscheidungsproblemen ohne Kiteriengewichtungsinformationen?
- SMAA erfordert kontinuierliche, reale Eingaben für Pref/Jugde
- Analyse basiert auf der Betrachtung eines Teilraums
- Rechenmodell besteht aus Integralausdrücken im Hochdimensionalen Raum, was nicht ausgewertet werden kann
-> Monte-Carlo-Simulation(10.000 Instanzen zufällig auswählen(Teilraum))
- durch zählen der Ergebnisse der Instanzen entsteht rank acceptability index
